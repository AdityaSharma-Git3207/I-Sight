# I-Sight 

I-Sight is a modern eye-care platform website built to make vision care simple, trusted, and accessible.

It combines doctor consultations, digital prescriptions, eyewear discovery, awareness resources, and NGO-supported accessibility into one seamless platform.

---

##  Features

- Modern responsive landing page
- Clean premium UI/UX design
- Eye consultation services showcase
- Eyewear marketplace section
- Digital health records concept
- NGO support & accessibility focus
- Smooth animations and interactions
- Fully responsive for desktop/mobile

---

##  Built With

- React.js
- Vite
- CSS3
- JavaScript (ES6+)
- Framer Motion
